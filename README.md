What is Galax Graphy?
=====================
Galax graphy is a note taking app for one's who need fast accessbility, and other features which traditional note taking apps lack. 

WORKSPACE
---------
1. General Workspace page for major seperation of knowlege spaces. 
2. Youtube Workspace for saving favourite vids categorised. 
3. Grouping and general necessary tools that other similar applications has. 

QUICK-ACCESS
------------
CTRL+SPACE -> Root directory of workspaces.
ALT+SPACE  -> Global search

APPEAREANCE
-----------
- Uses Super clean Minimal fonts and clear contrast colors.
- Simplicity first.

USAGE
-----
- Cloudflare worker with supabase.
